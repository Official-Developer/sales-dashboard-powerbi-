# Sales Performance Dashboard — Power BI

## Overview
Interactive Power BI dashboard to monitor key sales KPIs.

## Tech Stack
- Power BI Desktop
- DAX, Power Query
- Excel (Data Source)

## Features
- Total Sales, Orders, Average Order Value
- Region & Product wise drill-down
- Monthly trend analysis
- Time Intelligence measures
